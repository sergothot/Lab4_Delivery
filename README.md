# Лабораторная работа 3 C#: Управление заказами в службе доставки

Разработайте систему для управления заказами в службе доставки еды. 

Система должна:

1. Поддерживать создание и управление заказами, включая возможность выбора блюд из меню.
2. Обрабатывать различные типы заказов, такие как стандартные заказы и заказы с особенными условиями (например, быстрая доставка, персональные предпочтения).
3. Предоставлять возможность отслеживания состояния заказа: подготовка, доставка, выполнен.
4. Включать функции для расчета стоимости заказа с учетом различных факторов, таких как скидки, налоги, плата за доставку.
5. Логику необходимо покрыть unit тестами
6. CLI не нужен
7. Использовать не менее 5 паттернов ООП, изученных за курс. Иметь четкое обоснование почему был выбран каждый

Вы разрабатываете систему для стартапа в сфере доставки еды, который хочет обеспечить быструю и надежную обработку заказов. Система должна быть масштабируемой, чтобы справляться с растущим числом пользователей.

Цель ЛР заключается в изучении и применении паттернов проектирования для создания гибкой и расширяемой архитектуры, особенно в условиях, когда требования могут быстро изменяться.
